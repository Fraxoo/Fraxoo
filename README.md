## 💫 About Me

Développeur web en formation, je conçois et développe des projets full-stack avec **React, Node.js, JavaScript, TypeScript et PHP** afin de renforcer mon portfolio et mes compétences.

J’aime travailler sur des projets concrets, structurés et utiles, en cherchant toujours à améliorer :
- la qualité du code
- l’ergonomie des interfaces
- la logique applicative
- l’organisation globale d’un projet

Je progresse principalement sur la création d’applications web modernes, responsives et maintenables, du front-end jusqu’au back-end.
